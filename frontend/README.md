# AdminDasboard Frontend

Next.js App Router frontend for AdminDasboard.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

The development server runs at:

```text
http://localhost:3000
```

The frontend expects the local backend at:

```text
http://localhost:5160
```

Configured through:

```text
NEXT_PUBLIC_API_BASE_URL
```
